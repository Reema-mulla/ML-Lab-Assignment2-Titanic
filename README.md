# Assignment 2 – Machine Learning Lab

## Dataset Description
This project uses the Titanic dataset from Kaggle.  
The dataset contains information about passengers such as age, gender, passenger class, fare, and survival status.

The dataset includes 891 rows and 12 columns.

## Machine Learning Problem
This is a classification problem.  
The goal is to predict whether a passenger survived the Titanic disaster based on the available features.

## Repository Contents
- `data/train.csv` – Dataset file  
- `notebooks/titanic_eda.ipynb` – Jupyter Notebook showing basic dataset information  
- `methodology/` – Methodology diagram
